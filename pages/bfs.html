Breadth First Search (BFS)

BFS explores all neighbors of a node before moving to the next level. It's useful for level-wise traversal and shortest path in unweighted graphs.

How BFS Works:

Breadth First Search (BFS) starts from the root node and explores all its neighbors before going to the next level. It uses a queue data structure and is ideal for finding shortest paths in unweighted graphs.
